# twelve-factor-app

Reference guide for the Twelve-Factor App methodology — 15 principles (12 original plus 3
modern extensions) for building portable, resilient, cloud-native applications.

## What it does

Provides comprehensive coverage of all fifteen factors with guidance on applying each to
modern cloud-native environments. Incorporates the 2025 open-source community evolution,
Kubernetes and container reinterpretations, and GitOps workflow patterns. Use it when
evaluating application architecture, reviewing codebases for methodology compliance, or
advising on configuration, scaling, observability, security, and deployment patterns.

## Skills

- `twelve-factor-app` — Full reference for all 15 principles with cloud-native context and
  compliance guidance

## Installation

```bash
/plugin install twelve-factor-app@jamie-bitflight-skills
```

## Usage

```text
/twelve-factor-app "review this service's configuration handling against Factor III"
/twelve-factor-app "explain Factor XII (admin processes) for a Kubernetes deployment"
```
