# MkDocs

Comprehensive guide for creating and managing MkDocs documentation projects with Material theme. Includes official CLI command reference, mkdocs.yml configuration reference, and Material theme customization patterns.

## Installation

**From Marketplace:**

```bash
/plugin marketplace add Jamie-BitFlight/claude_skills
/plugin install mkdocs@jamie-bitflight-skills
```

**For Development:**

```bash
claude --plugin-dir /home/user/claude_skills/plugins/mkdocs
```

## Quick Start

Activate the mkdocs skill when working with MkDocs documentation projects:

```text
@mkdocs
Help me initialize a new MkDocs project with Material theme and configure it for a Python library
```

The skill provides:

- **CLI Reference**: Complete mkdocs command documentation (build, serve, new, gh-deploy)
- **Configuration Reference**: All mkdocs.yml settings and valid values
- **Material Theme Reference**: Customization options, features, and integrations
- **Plugins Reference**: Common plugin configurations (search, git-revision-date, minify)
- **Real-World Examples**: Production-ready configurations and patterns
