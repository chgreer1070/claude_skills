## Using Reference Materials

### Progressive Disclosure Strategy

The skill uses progressive disclosure to optimize context usage:

1. **Initial Request**: SKILL.md loaded (153 lines) - provides overview and pattern categories
2. **Pattern Selection**: Load pattern-selection-guide.md (40 lines) if choice unclear
3. **Pattern Application**: Load pattern-categories-and-documentation.md sections as needed
4. **Domain Specificity**: Load domain-specific-applications-and-variations.md if domain-specific templates needed
5. **Deep Research**: Load original source files (itonics, vanderbilt, etc.) for detailed research

### Recommended Reading Paths

**For Quick Ideation:**
1. SKILL.md → pattern-selection-guide.md → comprehensive-prompt-library-ready-to-use-templates.md

**For Understanding Patterns:**
1. SKILL.md → pattern-categories-and-documentation.md → synthesis-what-makes-these-patterns-work.md
