# dark.design

## Overview

**dark.design** is a curated gallery of hand-picked dark-themed websites, described as "the best dark themed websites on the internet. From designers for designers. Curated with love." The platform serves as both a design inspiration resource for web professionals and a marketplace for Framer-based website templates. Created and operated by **Y5K**, a design-focused product studio founded by Cédric Moore and based in Eastern Switzerland, dark.design combines design curation with template commerce to support the dark mode design community.

## Problem Addressed

Web designers seeking contemporary examples and best practices for dark-themed user interfaces face fragmentation across multiple sources — portfolio sites, design platforms, and agency showcases scattered across the internet with no unified directory. dark.design solves this by aggregating hand-picked dark-themed website examples in a single, searchable, categorized location. This enables designers to quickly find inspiration, study implementation patterns across industries, and discover best practices for dark mode design without manual research across multiple platforms.

## Technical Architecture

**Core Stack:**
- **Frontend:** Built with Framer (no-code website builder and design tool)
- **Membership/Payments:** Powered by Outseta (membership and subscription platform)
- **Organization:** Y5K (parent design studio and product company)

**Supporting Services:**
- Y5K operates three complementary businesses:
  1. **Trueform** — Design and development agency services
  2. **Canvas** — Framer templates and components library
  3. **Dark Website Curation** — The dark.design platform itself
- Integration with Framer Gallery ecosystem for template distribution and discovery
- Partner integrations with software vendors (Framer, Loops, etc.) for member deals

**Data Flow:**
Visitors navigate the curated directory (indexed by category), preview websites via mobile/desktop breakpoints, and may save bookmarks. Membership is managed through Outseta. Template purchases flow through Framer's marketplace infrastructure. Quarterly newsletter is distributed via email to paid members.

## Installation & Usage

**Accessing dark.design:**

1. Visit <https://www.dark.design/> in any modern web browser
2. Browse the free directory or categories (Software, AI, Commerce, etc.) without account
3. View featured sites with mobile and desktop previews
4. Optional: Create account to save bookmarks

**Upgrading to Paid Membership:**

1. Navigate to <https://www.dark.design/pricing>
2. Select "Start membership" for $19/year
3. Unlock features: full directory access, mobile/desktop previews, bookmarks, quarterly top-10 newsletter, member deals
4. Cancel anytime — no lock-in period

**Using Templates:**

1. Visit <https://www.dark.design/templates>
2. Browse available Framer templates (Display, Stream, CourseOS, Supply, etc.)
3. Click template to preview and purchase through Framer marketplace
4. Import into Framer workspace and customize for your project

**Submitting Content:**

- Design agencies can submit dark-themed websites for curation consideration (invitation-only; submit via contact on main site)
- Template creators can list templates for sale through "List a template for sale" option

## Relevance to Claude Code Development

dark.design is **moderately relevant** to Claude Code development in the following contexts:

1. **Design Inspiration for AI Tool UX** — The curated collection of dark-themed sites provides contemporary reference implementations for designing dark-mode interfaces in AI coding tools. Cursor (an AI code editor) is featured as a gallery example, demonstrating how dark mode design applies to developer-facing AI products.

2. **Template-Driven Prototyping** — Framer templates from dark.design can accelerate UI prototyping for Claude-related products, marketing sites, or documentation portals, reducing design-to-code iteration time.

3. **Design System Reference** — The categorized breakdown (Software, AI, Commerce) and featured examples (Linear, Cursor, Framer Studio) serve as a design system reference for applying dark-mode patterns consistently across Claude Code extensions and related tooling.

4. **Community Patterns** — The emphasis on "from designers for designers" and curation ethos aligns with Claude Code's design philosophy of providing thoughtful, well-crafted developer experiences.

**Limitations:**
- No API or programmatic access to the directory; curation is manual and human-driven
- Templates are Framer-specific; limited applicability to non-Framer projects
- Not a technical resource — focuses on design aesthetics and patterns, not architecture or implementation details
- No public data export or bulk curation data available
