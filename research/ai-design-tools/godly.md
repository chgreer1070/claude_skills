# Godly: Astronomically Good Web Design Inspiration

## Overview

Godly is a hand-picked curation of over 1,000 of the best web design examples on the internet, launched in July 2021 by designer Daryl Ginn and curated by Rejiggle, a conversion-focused design studio. The platform serves as a creative feed and inspiration gallery for designers, developers, and creative professionals seeking high-quality, bold, and experimental web design. Rather than offering templates or cloneable projects, Godly operates as a selective discovery platform that prioritizes quality over quantity, with each featured website vetted for exceptional interaction design, typography, and visual polish.

## Problem Addressed

Web designers and developers often struggle to find collections of high-quality, cutting-edge websites that push beyond conventional design patterns. Generic design inspiration sources frequently include mediocre work, corporate templates, and safe middle-ground designs that don't challenge creative thinking. Godly addresses this gap by providing a curated, high-signal source of unapologetically bold web design that emphasizes experimental and innovative approaches to digital experiences.

## Key Features

### Curated Gallery with Multiple Categories
Godly organizes featured websites across structured categories including agency websites, personal portfolios, SaaS products, e-commerce sites, Web3 projects, and AI tools. Users can browse curated collections filtering by design style, industry, or interaction type. The gallery emphasizes experimental studios and indie makers, positioning itself as a resource for bold design rather than corporate best practices.

### User Profile & Attribution System
Users can create public profiles that display any websites they created that were featured on Godly, track their own submissions, and curate personal collections. This attribution system credits designers and developers while allowing creators to build visibility and portfolio presence through platform features.

### Daily Feed Updates
The platform publishes multiple featured websites per day, providing a continuous stream of fresh design inspiration. This regular update cadence positions Godly as a daily resource for designers seeking current examples of innovative web design approaches.

### Collection Curation
Users have the ability to create and save personal collections of featured websites, enabling custom organization and reference for specific projects or design interests.

## Technical Architecture

**Access Method**: Web-based browser interface with no required authentication for basic browsing. Users may optionally create accounts to save collections and submit websites.

**Content Model**: Godly maintains a manually curated database of featured websites, with each entry including the website URL, design categorization, maker attribution, and potential metadata about key design elements (interactions, typography, etc.).

**Future Architecture Enhancements**: Planned features include website video playback to display interactions and animations, commenting systems for community discussion, and expanded user profile functionality to showcase maker portfolios and submission history.

## Installation & Usage

Godly is entirely browser-based with no installation required.

**Basic Access**:
1. Navigate to <https://godly.website> in any modern web browser
2. Browse featured websites by scrolling the main feed or using category filters
3. Click any website card to visit the featured site

**For Designers Seeking Specific Types of Inspiration**:
- Use category navigation to filter by site type (portfolio, e-commerce, SaaS, AI, Web3)
- Review featured examples from experimental studios and indie makers known for pushing design boundaries
- Examine typography, interaction patterns, color usage, and visual hierarchy in featured sites

**For Creators Submitting Work**:
1. Create a user account on Godly
2. Submit websites you've designed via the submission interface
3. Featured submissions appear on your public profile
4. Accepted submissions gain visibility within the Godly community

**Creating Personal Collections**:
1. Log into your user account
2. Browse the main feed or use category filters
3. Save websites to personal collections for future reference
4. Share collections via public profile

## Limitations and Caveats

- **No Cloneable Code**: Godly curates finished websites for visual and interaction reference only; it does not provide source code, component libraries, or reproducible implementations
- **Manual Curation Only**: Featured websites are subjectively selected by the Godly editorial team; coverage is not exhaustive and leans toward experimental and indie-maker work rather than corporate design patterns
- **Limited Real-Time Content**: While Godly publishes multiple websites weekly, it is not a real-time design discovery service and does not aggregate updates from all websites across the web
- **Emerging Features**: Commenting functionality, website video playback of interactions, and enhanced community features are planned but not yet implemented as of this research (2026-04-03)
- **Attribution Uncertainty**: Not all websites featured on Godly may be correctly attributed; the curator relies on submissions and public information
- **Interaction Documentation**: The platform plans to add video playback to showcase interactions and animations, but detailed written documentation of interaction mechanics is not currently standard
