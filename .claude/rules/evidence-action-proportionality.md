# Evidence-Action Proportionality

The number of files modified must be proportional to the evidence gathered.

One hypothesis confirmed by one test → fix ONE file maximum. Edits across multiple files/plugins require:

1. Root cause confirmed by reproduction in actual environment
2. Each affected file independently verified as exhibiting the issue
3. Fix validated on at least one file before applying to the rest

"This same fix applies to all N files" is a CLAIM requiring verification, not a conclusion that follows from fixing one file.
